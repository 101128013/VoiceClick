TROUBLESHOOTING
══════════════════════════════════════════════════════════════════════

Widget won't appear?
  1. Check it's not hidden - press ESC to toggle
  2. Restart: python monitor_control.py stop
  3. Then: python monitor.py

PyQt6 not found?
  pip install PyQt6

Progress not updating?
  1. Check file exists: C:\Users\SUPER\.voice_click\progress.json
  2. Edit with valid task number (1-30)
  3. Check logs for errors

Auto-start not working?
  Run: enable_autostart.bat again
  Check: type C:\Users\SUPER\.voice_click\monitor.log

Want to disable auto-start?
  Double-click: disable_autostart.bat
