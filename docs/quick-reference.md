# VoiceClick Quick Reference

Quick reference card for VoiceClick features and shortcuts.

## Keyboard Shortcuts

| Action | Default Shortcut |
|--------|-----------------|
| Start Recording | Ctrl+Shift+R |
| Stop Recording | Ctrl+Shift+S |
| Customize | Settings → Hotkeys |

## Main Features

- **Voice-to-Text**: Speak and text appears automatically
- **Auto-Start**: Begin recording when text field is focused/clicked
- **History**: View and search past transcriptions
- **Settings**: Customize model, device, hotkeys, and more

## File Downloads

- **Installer**: `VoiceClick-Setup-v1.0.0.exe` (~60-120MB)
- **Portable**: `VoiceClick-Portable-v1.0.0.zip` (~50-100MB)

## Installation Time

- **Installer**: ~2 minutes
- **Portable**: ~30 seconds
- **First Launch**: 2-5 minutes (model download, one-time)

## Settings Quick Access

- **Model**: Settings → Transcription Model
- **Device**: Settings → Compute Device
- **Hotkeys**: Settings → Hotkeys
- **Auto-Start**: Settings → Auto-Start Options

## Common Issues

| Issue | Solution |
|-------|----------|
| No audio | Check microphone permissions |
| Text not appearing | Ensure text field is active |
| Model download fails | Check internet connection |
