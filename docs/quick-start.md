# VoiceClick Quick Start

Get started with VoiceClick in minutes!

## Installation

1. Download `VoiceClick-Setup-v1.0.0.exe` from GitHub Releases
2. Run the installer
3. Launch from Start Menu

## First Use

1. Wait for model download (2-5 minutes, one-time only)
2. Click "Start Recording"
3. Speak into your microphone
4. Text appears automatically!

## Quick Tips

- **Start Recording**: Ctrl+Shift+R
- **Stop Recording**: Ctrl+Shift+S
- **View History**: History tab
- **Change Settings**: Settings tab

## Common Tasks

### Record and Transcribe
1. Click "Start Recording"
2. Speak
3. Text appears in active field
4. Click "Stop Recording"

## Troubleshooting

**No sound?** Check microphone permissions in Windows Settings.

**Text not appearing?** Make sure a text field is active/focused.

**Model download slow?** Normal on first launch. Subsequent launches are instant.
